# Portofolio Oji Syaputra
## Isi File

- `index.html` — struktur halaman website
- `style.css` — tampilan visual bertema teknologi
- `script.js` — menu responsif dan tahun otomatis
- `assets/favicon.svg` — ikon website
- `assets/oji-syaputra-photo.jpeg` — foto profil

## Cara Upload ke GitHub Pages

1. Buat repository dengan nama:
   `syaputra94.github.io`

2. Upload semua file dari ZIP ini ke repository tersebut.

3. Masuk ke:
   **Settings → Pages**

4. Pada bagian **Build and deployment**, pilih:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

5. Klik **Save**.

6. Tunggu beberapa menit, akses di Link :
   `https://syaputra94.github.io`
